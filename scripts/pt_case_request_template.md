## 3. 받은 내용을 JSON으로 변환하는 방법

PT에게 받은 내용을 아래 형식으로 옮겨서 `scripts/token_compare.py`에 넣으면 됩니다.

```json
[
  {
    "case": "케이스 1 — SOAP 노트",
    "task": "오늘 세션 SOAP 노트 작성해줘.",
    "prompt_a": "PT가 입력한 내용 그대로",
    "prompt_b": "Sage Pontus로 생성한 압축 프롬프트"
  },
  {
    "case": "케이스 2 — HEP",
    "task": "홈 운동 프로그램 만들어줘.",
    "prompt_a": "PT가 입력한 내용 그대로",
    "prompt_b": "Sage Pontus로 생성한 압축 프롬프트"
  }
]
```

**변환 팁:**
- `task`는 PT가 쓴 문장에서 "핵심 질문" 부분만 추출
- `prompt_a`는 task를 제외한 배경 설명 전체
- `prompt_b`는 Sage Pontus 질문 플로우를 PT 정보로 채워서 생성
